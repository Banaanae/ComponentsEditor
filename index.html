How to use

  1. Enter the item you want to create in the search box above
  2. On the right side, click on the component's of the item you want to edit, you can click the Minecraft wiki reference for more information on each component
    • Ticking the "Active" checkbox makes the component get added to the command
    • Ticking the "Remove" checkbox removes that component (if default behaviour) from the item
    • All component's inputs are the same as found in the wiki, if not there will be a symbol which will give more information when hovered
  3. Click generate command to use in Minecraft, note that commands longer than 256 characters need to be entered via command block
